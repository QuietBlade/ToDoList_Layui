### 基于layui的 事项清单

使用localStroe作为缓存

为了轻量化项目，使用了CDN

可以通过file协议直接访问

### 系统功能

+ 使用localStore存储，数据不丢失
+ 编不出来了

### 功能清单

- [x] 添加事项
- [x] 删除事项
- [x] 修改事项
- [x] 完成/未完成事项
- [x] 分页事项

### 缺点

+ 没有设计时间
+ 新建事项会在底部
+ 在 未完成/已完成 页面添加事项会跳转到 所有事项 页面
